# AI 레벨 테스트 (AI Level Test)

AI 상식 및 활용 숙련도 레벨을 확인하는 정적 웹 프로젝트(HTML/JS/CSS)입니다.

## 실행 방법

이 프로젝트는 정적 파일들로 구성되어 있어 별도의 빌드 과정 없이 바로 실행할 수 있습니다.

### 1. 브라우저로 직접 열기
프로젝트 폴더 내의 `index.html` 파일을 더블 클릭하여 브라우저에서 엽니다.

### 2. 로컬 서버로 실행하기 (권장)
Node.js가 설치되어 있다면 아래 명령어로 로컬 서버를 띄워 확인할 수 있습니다.
```bash
npx http-server .
```

## 배포 시 주의사항
- **도메인 설정**: `CNAME` 파일이 포함되어 있습니다. 배포 시 해당 도메인 연결 설정이 유지되는지 확인해주세요.
- **광고**: `ads.txt` 파일이 포함되어 있습니다. 배포 루트 경로에 함께 업로드되어야 합니다.
